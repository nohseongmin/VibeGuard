## 2. 사용자가 직접 해야 할 일

### 2-1. 시연영상 (필수)

가장 쉬운 방법 — 자동 재생 데모를 화면 녹화 (도구 조작 불필요)
1. `demo/demo.html` 을 더블클릭해 브라우저로 연다. (F11 로 전체화면)
2. 녹화 시작: 키보드 `Win + Alt + R` (Windows 11 기본 게임바 녹화). 또는 OBS·파워포인트 녹화.
3. 화면의 ▶ 를 클릭하면 약 2분간 자동으로 재생된다(자막 포함). 끝까지 둔다.
4. 다시 `Win + Alt + R` 로 정지. 영상은 `동영상\캡처` 폴더에 저장됨.
5. (선택) 재생되는 동안 아래 대본을 소리 내어 읽으면 목소리 해설이 들어간다. 안 읽어도 자막으로 설명됨.
6. 유튜브에 업로드(공개/일부공개) 후 URL 을 보고서 '시연영상' 칸에 기재.

직접 시연하고 싶다면 — 아래 대본대로 진행
- 대본: `보고서/시연영상_대본.md` — 장면별 나레이션·입력 명령어·GUI 시연 포함.
- 녹화: 화면 녹화 도구(OBS, PowerPoint 녹화, Windows Game Bar 등)로 터미널과 브라우저를 캡처. 목표 길이 약 2분 30초~3분.
- 업로드: 유튜브에 업로드(공개 또는 일부공개). 별도 영상 파일 제출은 불가, URL만 기재.
- 기재: 보고서 '시연영상' 칸의 "(제출 전 시연영상 유튜브 URL 기재 예정)"을 실제 URL로 교체.
  - 방법: DOCX를 Word로 열어 해당 칸을 수정한 뒤, "다른 이름으로 저장 → PDF"로 PDF를 다시 내보냅니다.

## 3. 도구 실행 방법 (시연·심사 참고)

설치:
```
git clone https://github.com/nohseongmin/VibeGuard
cd VibeGuard
pip install -e .
```

주요 명령:
```
vibeguard scan examples/vibe_coded_app          # 데모 취약앱 스캔(취약점 다수 + 슬롭스쿼팅 탐지)
vibeguard gui                                   # 브라우저 GUI (http://127.0.0.1:8000)
vibeguard scan . --format html -o report.html   # 단독 HTML 리포트
vibeguard scan . --format sarif -o out.sarif    # SARIF(GitHub 코드 스캐닝/VS Code)
vibeguard rules                                 # 탑재 규칙 42개 목록
```

요구사항: Python 3.8 이상 (런타임 외부 의존성 없음 — 표준 라이브러리만 사용).
테스트: `pip install -e ".[dev]"` 후 `pytest` (69개 통과).
